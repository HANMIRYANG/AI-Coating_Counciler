# Claude Design Request Prompt

디자인 요청 대상:
AI Coating Council - 기능성 특수도료 AI 기술검토 회의 시스템

목표:
Gemini, Claude, GPT가 각자 의견을 내고, 이후 회의/상호비판을 거쳐 최종 답변을 만드는 B2B 기술검토 UI를 디자인해주세요.

디자인 요구사항:
- 단색 중심
- 텍스트 가독성 최우선
- 고대비
- 부장님/기술영업 담당자가 이해하기 쉬운 화면
- 일반 생성형 AI처럼 채팅형 입력
- 그러나 결과 화면에서는 AI별 의견과 회의 과정이 명확히 보일 것

주요 화면:
1. 메인 채팅 입력 화면
2. Round 진행 화면
   - Round 1: AI별 독립 의견
   - Round 2: 회의/상호비판
   - Round 3: 최종 합의안
3. 최종 결과 화면
   - 업체 발송용 답변
   - 내부 검토용 메모
   - 누락 자료
   - 위험 표현
   - AI별 초안
   - 회의 결과

색상:
- 기본은 단색 또는 1개 포인트 컬러
- 너무 많은 색상 금지
- 상태 표시는 색상뿐 아니라 텍스트/아이콘으로도 구분

산출물:
- 메인 화면 디자인
- 진행 상태 화면 디자인
- 결과 화면 디자인
- 카드 컴포넌트 디자인
- 디자인 토큰
- 개발자가 React/Tailwind로 옮기기 쉬운 설명
