# Risk Phrase Checklist

## 금지 또는 경고 표현

```text
완전 방지
100% 안전
불에 타지 않음
화재를 막음
폭발 방지
열폭주 방지
인증 완료
법적으로 문제 없음
모든 소재 적용 가능
반영구적
무조건 효과
검증 완료
세계 최고
```

## 대체 표현

```text
특정 시험 조건에서 확인 필요
화염 확산 지연 가능성
기재 보호 보조 효과
도포 조건에 따라 성능 상이
추가 시험 필요
인증기관 확인 필요
자료 기준으로 제한적 검토 가능
```

## 자동 경고 조건

- 위험 표현이 businessReadyAnswer에 포함됨
- 인증/법령 단정 표현 포함
- 시험성적서 없이 성능 보증
- 배터리 화재 방지 단정
- 폭발 방지 단정
